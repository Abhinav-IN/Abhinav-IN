## Projects

### SoulSync — Dating Application with Microservices
> Polyrepo architecture · 7 NestJS services · Kafka event bus · WebSocket · Spotify OAuth 2.0 · Self-hosted VPS

Built a full-stack dating app where match scores are computed using **Jaccard similarity** on users' Spotify top artists and tracks. Services communicate asynchronously via Kafka. Deployed across 8 independent repositories on a self-hosted VPS via Coolify.

[![Repo](https://img.shields.io/badge/View%20Architecture%20%26%20README-6C63FF?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Abhinav-IN/soul-sync)
[![Live Demo](https://img.shields.io/badge/Live%20Demo-6C63FF?style=for-the-badge)](https://soulsync.manavkashyap.com/demo)

**Stack:** NestJS · Kafka (Aiven) · MySQL (TiDB) · MongoDB Atlas · Cloudinary · Spotify API · React
